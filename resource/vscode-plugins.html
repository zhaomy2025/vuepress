VS Code 插件合集

  • 界面美化与视觉增强
  • 代码结构与导航
  • 编码效率提升
  • 代码质量与格式化
  • 开发工具与辅助功能
  • Git与版本控制
  • 代码片段类插件

提示

Visual Studio Code 作为一款轻量级但功能强大的代码编辑器，其丰富的插件生态系统是它最大的优势之一。本文精心整理了一系列能够显著提升开发效率、美化界面、优化代码质量的VS Code插件，涵盖界面美化、代码导航、效率提升、质量保障等多个方面，帮助你打造专属的理想开发环境。

界面美化与视觉增强

  1. Better Comments
    美化注释的插件，可根据不同种类的注释显示不同颜色，支持自定义颜色和注释类型。

  2. Community Material Theme
    编辑器主题，提供多种编辑器主题供选择。

  3. Material Icon Theme
    文件图标主题插件，内置多种图标，安装完了以后点击设置文件图标主题就可以了。

  4. VSCode Great Icons
    另一个文件图标主题插件。

  5. Image preview
    鼠标悬停在代码中的图片引用上时显示图片预览。

  6. indent-rainbow
    为不同缩进层级显示不同颜色。

  7. Indenticator
    点击缩进部分时显示指示线，帮助查看代码结构。

  8. Trailing Spaces
    高亮显示代码中的尾随空格。

代码结构与导航

  1. Bracket Pair Colorizer
    为配对括号显示相同颜色，并在选中括号时显示连接线指示配对关系。

  2. Highlight Matching Tag
    高亮显示配对的HTML标签。

  3. Color Highlight
    直观显示代码中设置的颜色值。

  4. CSS Peek
    通过点击类名快速定位到样式定义。

编码效率提升

  1. Auto Import
    自动导入TypeScript模块。

  2. Auto Rename Tag
    重命名开始标签时自动重命名配对的结束标签。

  3. change-case
    快速切换变量命名格式（大驼峰、小驼峰、下划线等）。

  4. ECMAScript Quotes Transformer
    在模板字符串和普通字符串拼接之间相互转换。

  5. embrace
    快速在选中代码两侧添加引号、括号等符号。

  6. File Utils
    提供创建、复制、移动、重命名、删除文件和目录的便捷操作。

  7. javascript console utils
    快速生成和删除console.log调试语句。

  8. json2ts
    将JSON格式自动转换为TypeScript类型定义。

  9. koroFileHeader
    自动添加文件头部注释和函数注释，支持自定义配置。

  10. Mithril Emmet
    快速生成代码结构。

  11. Path Intellisense
    引入文件时提供路径自动补全功能。

  12. Npm Intellisense
    导入npm包时提供智能提示。

  13. px to rem & rpx (cssrem)
    自动进行像素单位换算。

  14. Turbo Console Log
    支持自定义格式的console.log生成工具。

代码质量与格式化

  1. ESLint
    代码检查工具，帮助发现并提示不符合规范的代码。

  2. Prettier - Code formatter
    代码格式化工具，支持自定义格式化规则。

开发工具与辅助功能

  1. Code Runner
    在编辑器中直接运行代码并查看结果。

  2. CodeIf
    通过搜索GitHub、Bitbucket、GitLab提供变量命名建议。

  3. Color Info
    显示颜色的详细信息，包括RGB、HSL、CMYK、HEX等格式。

  4. Code Spell Checker
    检查代码中单词拼写，支持自定义词典。

  5. Debugger for Chrome
    在VS Code中直接进行Chrome浏览器调试。

  6. Local History
    保存本地代码修改记录，支持版本对比。

  7. open in browser
    在浏览器中快速打开HTML文件。

  8. Partial Diff
    比较代码片段差异的工具。

  9. Postcode
    VS Code内的API测试工具，类似Postman功能。

  10. Project Manager
    项目管理工具，方便快速切换多个项目。

  11. Quokka.js
    实时显示代码运行结果。

Git与版本控制

  1. Git History
    以列表形式查看文件的Git提交记录。

  2. GitLens --- Git supercharged
    增强Git功能，显示代码行的提交信息。

代码片段类插件

提供常用代码片段，通过简写快速生成完整代码结构。

  1. JavaScript (ES6) code snippets
    提供ES6语法的JavaScript代码片段。
  2. Jest Snippets
    提供Jest测试框架的代码片段。
  3. HTML Snippets
    提供HTML代码片段。
  4. Vue VSCode Snippets
    提供Vue框架的代码片段。
  5. Vue 3 Snippets
    提供Vue 3框架的代码片段。
