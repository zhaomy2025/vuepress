Zookeeper命令行

  • 服务端
    • 启动Zookeeper

    • 打开客户端

服务端

启动Zookeeper

zkServer.sh start

停止Zookeeper

zkServer.sh stop

客户端

打开客户端

zkCli.sh
zkCli.sh -server host:port

退出客户端

quit

创建节点

create [-s] [-e] path data acl

提示

-s：顺序节点，会按照路径名的字母顺序在子节点中排列； -e：临时节点，会话失效则节点自动删除。

# 创建zk-temp临时节点
create -e /zk-temp 123
# 创建永久节点
create /zk-perm 123

读取节点

# 列出Zookeeper指定节点下的所有子节点
ls path [watch]
# 查看指定节点下的第一级的所有子节点
ls2 path [watch]
get path [watch]

更新节点

set path data [version]

删除节点

delete path [version]

提示

若删除节点存在子节点，那么无法删除该节点，必须先删除子节点，再删除父节点。
