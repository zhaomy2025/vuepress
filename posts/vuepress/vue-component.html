编写 Vue 组件

VuePress 编写 Vue 组件步骤如下：

  1. 创建自定义组件
  2. 手动全局注册组件(可选)
  3. 在 Markdown 中使用自定义组件

创建自定义组件

在 .vuepress/components/ 目录下创建 MyComponent.vue 文件，内容如下：

<template>
  <div class="my-component">
    <h2>这是一个自定义组件</h2>
    <p>这是组件的内容。</p>
  </div>
</template>

<script>
export default {
  name: 'MyComponent'
}
</script>

<style scoped>
.my-component {
  border: 1px solid #eaecef;
  padding: 20px;
  margin: 1em 0;
}
</style>

注意：

  1. vuepress 1.X 和 2.X 版本的组件写法不同，请根据版本选择对应的写法，本文以 2.X 版本为例。
  2. VuePress 默认会自动注册 .vuepress/components/ 下的 Vue 组件，但需确保
    1. 组件文件名为 MyComponent.vue（大小写敏感）
    2. 路径必须是 .vuepress/components/，不能是其他路径，包括子目录。

手动全局注册组件(可选)

如果自动注册失败，可以在 .vuepress/client.js 中手动注册：

import MyComponent from './components/MyComponent.vue'

export default defineClientConfig({
    enhance({ app }) {
        app.component('MyComponent', MyComponent)
    },
})

在 Markdown 中使用自定义组件

<MyComponent></MyComponent>

行内标签组件

组件代码：

MyTag.vue
<template>
  <span class="my-tag inline-tag" :style="{ backgroundColor: color }">
    <slot />
  </span>
</template>

<script>
export default {
  props: {
    color: {
      type: String,
      default: '#3498db' // 默认蓝色 var(--hint-c-soft);
    }
  },
  name: 'MyTag'
}
</script>

<style>
.my-tag {
  color: white;
}
</style>
Tip.vue
<template>
  <span class="tip inline-tag">
    <slot />
  </span>
</template>

<script>
export default {
  name: 'Tip'
}
</script>

<style scoped>
.tip {
  background: rgb(16 185 129 / 14%);
  color: #18794e;
}
</style>
Warning.vue
<template>
  <span class="warning inline-tag">
    <slot />
  </span>
</template>

<script>
export default {
  name: 'Warning'
}
</script>

<style scoped>
.warning {
  background: rgb(234 179 8 / 14%);
  color: #915930;
}
</style>
Danger.vue
<template>
  <span class="danger  inline-tag">
    <slot />
  </span>
</template>

<script>
export default {
  name: 'Danger'
}
</script>

<style scoped>
.danger {
  background: rgb(244 63 94 / 14%);
  color: #b8272c;
}
</style>

公共样式放到了.vuepress/styles/index.scss 文件中：

// 行内标签
.inline-tag {
  display: inline-block;
  padding: 0.2em 0.6em;
  margin: 0 0.2em;
  border-radius: 3px;
  font-size: 0.85em;
  font-weight: bold;
}

在 Markdown 中使用：

<MyTag>自定义标签</MyTag> 
<Tip>提示</Tip> 
<Warning>注意</Warning> 
<Danger>警告</Danger>

总结

  • 博客相关

  • 新功能
  • 站点管理
  • Markdown 相关

  • 行内标签组件
