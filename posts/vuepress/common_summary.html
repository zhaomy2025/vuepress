总结

VuePress 市场和VuePress 生态系统提供了很多常用插件，包括：

  • 博客相关
  • 面向开发者
  • 搜索插件
  • 新功能
  • 站点管理
  • Markdown 相关

Markdown 相关插件：

  • Markdown 高亮器
  • Markdown 样式化：提示容器、图片尺寸、样式化内容等
  • Markdown 语法扩展：代码选项卡、公式、自定义容器等
  • 功能集成：集成图表、代码、幻灯片等
  • 辅助工具：vuepress-plugin-markdown-define2

优先使用官方插件，官方插件不支持的功能，再考虑自己编写插件或Vue组件来实现相关功能，Vue组件的开发可以参考VuePress 编写 Vue 组件，目前实现了以下功能：

  • 行内标签组件

  • 标题自动编号
