# SSH
[[toc]]
## 获取用户名、密钥
### 服务器用户名
服务器用户名通常为 `root` 或 `ubuntu`


### 服务器获取SSH密钥
服务器私钥需管理员授权（通常位于 /etc/ssh/ssh_host_*）


### 腾讯云服务器创建密钥
1. 轻量云
2.SSH密钥
3. 点击创建密钥
