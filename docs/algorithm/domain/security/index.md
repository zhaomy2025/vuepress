## [现代密码协议与应用](./modern-crypto-protocols.md)

现代密码协议是构建安全通信与数据保护体系的核心框架，将基础密码学原语（如加密算法、哈希函数、数字签名）有机结合，为各种应用场景提供端到端的安全保障。本部分内容将：

- 解析现代密码协议的核心概念与设计原则，涵盖安全目标、威胁模型与协议架构
- 深入探讨主流密码协议的技术原理与实现机制
  - IPsec：基于AH/ESP协议和IKE密钥交换的网络层安全通信标准
  - SSH：提供身份认证、密钥交换和加密通信的传输层远程安全访问协议
  - TLS/SSL：保护Web通信的传输层安全协议
  - OpenPGP：基于混合加密架构和Web of Trust信任模型的端到端加密通信标准
- 分析密码协议在不同场景下的应用实践
  - 安全Web通信（HTTPS）
    - TLS/SSL：加密HTTP通信的安全协议
  - 电子邮件加密与签名
    - S/MIME：基于X.509证书的商业电子邮件加密与签名标准
    - OpenPGP：基于Web of Trust模型的开源电子邮件加密与签名协议
  - 虚拟专用网络（VPN）
    > VPN本身是一组技术的统称，并非单一协议，其所属的网络协议层级取决于具体实现
    - SSL VPN：基于TLS/SSL的传输层VPN
    - IPsec VPN：基于IPsec的网络层VPN
    - WireGuard：基于现代密码学的轻量级网络层VPN
  - 安全远程访问
    - SSH：用于远程服务器管理
    - RDP：用于Windows远程桌面
    - VNC：用于跨平台远程桌面
    - ZTNA：基于身份的访问控制
- 探讨现代密码协议面临的安全挑战
  - 协议漏洞与攻击手段
  - 量子计算威胁与后量子密码协议
- 分享密码协议部署与使用的最佳实践

## [后量子密码学简介](./post-quantum-cryptography.md)

后量子密码学（Post-Quantum Cryptography, PQC）是一门研究抵抗量子计算机攻击的密码算法的学科。随着量子计算技术的快速发展，现有的公钥密码体系正面临前所未有的威胁，后量子密码学成为保障未来信息安全的关键技术。

本部分内容将：
- 分析量子计算对现有密码体系的威胁，包括Shor算法和Grover算法的影响
- 探讨后量子密码学的主要技术路线，如格基密码、哈希基密码、编码基密码等
- 介绍NIST后量子密码算法标准化进程及已选定的标准算法
- 分析后量子密码学在TLS、区块链、物联网等领域的应用前景
- 提供后量子密码学迁移策略与最佳实践指南

通过学习后量子密码学，读者将了解如何为未来量子计算时代做好安全准备，确保数字资产的长期安全性。

|---------|---------|-----------------|---------|---------|---------|
| 格基密码学 | 格上最短向量问题(SVP)、最近向量问题(CVP)、学习错误(LWE)问题 | CRYSTALS-Kyber | CRYSTALS-Dilithium、FALCON、SPHINCS+ | CRYSTALS-Kyber | Kyber-Encrypt |

| 超奇异椭圆曲线同源密码学 | 超奇异椭圆曲线间同源的计算与逆向难度 | SIKE(已撤回) | - | SIDH、CSIDH | - |
