## 生成密钥
- 选两个特别的秘密质数，比如：
  - 秘密质数1：p = 3
  - 秘密质数2：q = 11

### 计算公钥(N, e)
- N = p * q = 3 * 11 = 33
- φ(N) = (p-1) * (q-1) = 2 * 10 = 20
- 选一个小的、随机的整数 e，满足 1 < e < φ(N) 且 e 与 φ(N) 互质。例如：e = 3
- 公钥就是 (N, e)，例如：(33, 3)

### 计算私钥（d）
- 计算 d，使得 (e * d) % φ(N) = 1
- (3 * d) % 20 = 1
- 计算可得：d = 7

> 注意：d 不是唯一的，一般会选择最小的那个。

## 使用
### 加密
- 假设你要加密的消息是 M = 4
- C = (M^e) % N。
- (4^3) % 33 = 31
- 加密后的消息 C = 31。

### 解密
- 假设你收到了加密后的消息 C = 31
- M' = (C^d) % N
- (31^7) % 33 = 31 * (31^2)^3 % 33 = 31 * 64 % 33 = 31 * 31 % 33 = 4
- 解密后的消息 M = 4

> (a * b) % c = (a%c) * (b%c) % c

## 解密证明

> 欧拉定理：对于任何与N互质的数M，有 (M^φ(N)) % N = 1

加密：C = (M^e) % N,
解密：M' = (C^d) % N
证明：M = M'

- 将 C = (M^e) % N 带入解密公式，可得：M' = [(M^e) % N]^d % N
- 根据(a * b) % c = (a % c) * (b % c) % c，可得：M' = (M^e)^d % N = M^(e * d) % N
- 私钥d的计算公式：e * d % φ(N) = 1，可得M' = M^(k * φ(N)+1) % N，其中k是一个整数
- 因此，解密公式为：M' = M^(φ(N))^k * M % N
- 根据欧拉定理，M^(φ(N)) % N = 1，因此：M' = M^(φ(N))^k * M % N = M % N
- 因为M小于N, 所以M' = M % N=M
