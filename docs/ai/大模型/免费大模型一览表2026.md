### 六、接入方式速查

无论是 Claude Code、Codex、Hermes 还是 OpenClaw，通过 **CC Switch**（[ccswitch.io](https://ccswitch.io/zh/)）都可以将免费模型接入 AI Agent。

**讯飞星火 Qwen3.6 接入 Claude Code：**

1. 注册讯飞开放平台，创建应用，复制 modelId 和 HTTP 协议 API Key
2. 下载 CC Switch，安装后选择 Claude Code
3. 创建自定义配置，填入 API Key
4. 请求地址填：`https://maas-api.cn-huabei-1.xf-yun.com/anthropic`
5. 实际请求模型填 modelId，保存启用
6. 测试连通后打开 Claude Code 即可使用

**Agnes AI 接入 Claude Code：**

1. 注册 Agnes API Platform（https://platform.agnes-ai.com/），创建 API Key
2. 安装 CC Switch，选择 Claude Code
3. 启用 Claude 相关路由
4. 新增供应商：类型 claude，请求地址 `https://apihub.agnes-ai.com/v1`，模型填 `agnes-2.0-flash`
5. 添加兼容参数避免请求失败：

```json
{
  "allowed_openai_params": ["thinking", "context_management"],
  "litellm_settings": {
    "drop_params": true
  }
}
```

> **通用小技巧**：Claude Code 的请求地址通常走 `/anthropic` 路径，其他 Agent 多走 `/v2` 路径。遇到参数不兼容时，在 CC Switch 中添加 `drop_params: true` 可避免请求失败。
