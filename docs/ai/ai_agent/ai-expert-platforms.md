# 天工、袋袋 Profy、QoderWake——AI 专家与数字员工平台对比

> 2026 年，AI 从工具进入"员工化"阶段。天工超级智能体在云端组建 AI 团队、袋袋 Profy 让真人经验变成可售卖的 AI 专家、阿里 QoderWake 让数字员工有岗位有记忆有治理。三者分别代表了工具型、市场型、治理型三条路线。

[[toc]]

---

## 一、AI 员工的三种形态

当 AI 不再只是"问一句答一句"的工具，而是能独立承担岗位职责时，就进入了"AI 员工化"阶段。目前国内有三条清晰的路线：

| 路线 | 代表产品 | 核心理念 |
|:---|:---|:---|
| **工具型** | 天工超级智能体（昆仑万维） | 打开浏览器就有一支云端 AI 团队，零部署、全模态 |
| **市场型** | 袋袋 Profy（未来式智能） | 真人专家封装经验上架市场，按调用付费 |
| **治理型** | QoderWake（阿里 Qoder） | 本地数字员工，有岗位有记忆可追踪可审批 |

三条路线不互斥，分别满足不同需求：天工解决"我直接要有 AI 团队"，袋袋解决"我要买专家的经验"，QoderWake 解决"我要管理 AI 员工"。

## 三、天工超级智能体——云端 AI 团队平台

### 3.1 产品背景

天工超级智能体由昆仑万维推出，定位为"云端 AI 团队平台"。与需要本地部署的 Agent 产品不同，天工**零部署、零配置**——打开浏览器注册即用。

### 3.2 核心能力

**6 大内置专家技能**：文档、PPT、图片、表格、网页、视频，自动搭配调用。

**内置顶级模型**：
- 主调度：DeepSeek V4 Pro、GLM-5.1、Kimi-2.6、MiniMax-2.7、Qwen3.6-Plus
- 视频生成：Seedance 2.0（世界第一视频生成模型）、可灵 3.0

**全模态编辑器**：图片可拆分元素编辑（文字/颜色直接改），视频有时间线剪辑（字幕/转场/滤镜/音频），网站可看源码改代码，PPT 可直接调整内容——所有产物不用导出到其他工具二次加工。

**多线程并行**：四个产物（小红书图文 + 视频 + 网站 + PPT）同时跑，而非串行排队。

### 3.3 实测案例

**repurpose-article Skill**：丢一篇公众号文章链接，天工自动执行以下步骤：
1. 读取文章内容，生成结构化骨架
2. 同时启动四个子任务——小红书图文、视频、可视化网站、PPT
3. 视频：拆分镜脚本 → Seedance 2.0 生成画面 → 语音模型配旁白 → 自动对齐时长 → 内置剪辑
4. 小红书：自动生成多张风格卡片，可直接编辑
5. 网站：自动部署，给出可访问 URL
6. PPT：7 页大纲 → pptx 生成，精准提取适合展示的内容

### 3.4 一句话定位

**天工 = 打开浏览器就有的云端 AI 团队，视频/图片/文档/网页全能，零门槛。**

## 四、袋袋 Profy——全球首个 AI 专家市场

### 4.1 产品背景

袋袋（Profy）由未来式智能（Agentspro.cn）推出，深耕智能体三年，此前已服务 120+ 头部企业。HLE（人类终极知识测验）中超越基模近 20 个百分点。

袋袋不是"又一个 AI 工具"，而是一个**双边市场**：专家端封装经验上架，买方端按需雇佣。

### 4.2 两种封装模式

**非技术专家**：自然对话即可。告诉袋袋遇到什么情况怎么处理、什么绝对不能碰，后台自动解析业务逻辑并封装。不需要学提示词、不需要写代码。

**技术大佬**：CLI 模式，把调校好的项目文件一行命令上传。

### 4.3 与 WorkBuddy 专家的本质区别

```
WorkBuddy 专家                    袋袋 Profy
┌─────────────────────┐           ┌─────────────────────┐
│ 平台养兵，用户调用    │           │ 专家养兵，市场交易    │
│                      │           │                      │
│ 腾讯预设 181+ 角色    │           │ 真人专家自行封装      │
│ 免费使用              │           │ 付费市场，收益归专家   │
│ 平台拥有所有权        │           │ 专家个人拥有（IP黑盒） │
│ 依赖平台手动更新      │           │ 五层自进化管线        │
└─────────────────────┘           └─────────────────────┘
```

### 4.4 两大底层壁垒

**IP 黑盒保护**：所有封装进去的核心逻辑在服务端密封运行，买方只能拿到成品，无法逆向破解或复刻。买的是鱼，不是渔。

**五层认知捕获管线**：L1 原始轨迹采集 → L2 信号提取 → L3 知识结晶 → L4 跨用户聚合 → L5 进化反馈。每次交互中的微调、纠偏和认可，自动转化为智能体的认知养分。专家今天上架的版本，恰恰是它这辈子最弱的版本。

### 4.5 典型案例

**风投人 Selene**：封装早期尽调经验后，意外发现 90% 的收益来自创业者用她的分身评估创业方案（而非同行做尽调同行）。

**公关专家宫琼**：品牌全球化专家分身成为 24 小时引流器，用户顺着分身找到本人签定制化全案。

### 4.6 一句话定位

**袋袋 = 把你的脑子打包成产品，卖给需要的人。收益归你，一次封装终身复利。**

## 五、QoderWake——阿里系的数字员工平台

### 5.1 产品背景

QoderWake 由阿里 Qoder 推出，是 Qoder 生态下的数字员工产品。在本地电脑（如 MacMini）上组建一支 7×24 小时持续工作的数字员工团队。

与天工（云端团队）和袋袋（专家市场）最大的差异在于：QoderWake 的核心关注点不是"AI 能力"而是"AI 治理"——数字员工的管理体系。

### 5.2 内置岗位

公测首发开放六种内置岗位：后端工程师、前端开发工程师、测试工程师、产品经理、数据分析师、内容运营专员。每种岗位经过系统性训练。也支持完全自定义角色。

每位数字员工有头像、入职时间、工作记录、热力图、记忆积累——不是用完即弃，而是越用越懂你。

### 5.3 治理能力（核心差异）

**长期记忆**：自动沉淀、整理，越用越懂项目偏好，不用每次从头教。可以查看每位员工最近学到的技能、进化的能力、更新的记忆。

**安全权限四维体系**：
- 工具防护：拦截命令注入和资源滥用
- 文件防护：精确控制每位员工对本地文件的读写权限
- 内置工具：可控可追溯
- 高危操作：触发审批，必须用户确认才能继续

**并行执行**：可不断安排新任务，每次并行推进而非串行排队。

**钉钉打通**：手机端召唤数字员工，接入内部文档、知识库和业务系统（通过 MCP）。

### 5.4 模型支持

GLM-5.1、DeepSeek V4 pro、Kimi-K2.6、MiniMax-M2.7、千问 3.7-max，支持 auto 模式让 AI 根据任务自动选择。Ultimate 背后是 Claude Opus 4.7。

### 5.5 待改进

- 任务交付成果不直观（存放在本地文件夹但页面上不展示，需手动找路径）
- 缺乏看板和任务管理列表（让数字员工按优先级、截止日期自动认领任务）

### 5.6 一句话定位

**QoderWake = 能在本地 7×24 运行的数字员工，有岗位有记忆有审批，治理能力优先。**

## 六、三款产品对比总览

### 6.1 路线对比

| 维度 | **工具型（天工）** | **市场型（袋袋）** | **治理型（QoderWake）** |
|:---|:---|:---|:---|
| 核心价值 | 零门槛的 AI 团队 | 专家经验可交易 | 数字员工可管理 |
| 部署 | 浏览器即用 | 云端平台 | 本地部署 |
| 用户角色 | 使用者 | 专家（供给方）+ 买方 | 管理者 |
| 内容形态 | 文档/PPT/视频/网页 全模态 | 特定专业领域 | 文档/代码/数据 |
| 变现 | 平台收费 | **专家收益** | 免费公测 |
| 适合谁 | 想直接用 AI 干活的人 | 有经验想变现的专家 | 想管好 AI 员工的人 |

### 6.2 选型建议

| 场景 | 推荐选择 |
|:---|:---|
| 零门槛直接拥有 AI 团队，需要多模态产出 | **天工超级智能体** |
| 有专业经验想变现，做数字分身 | **袋袋 Profy** |
| 需要专业领域咨询（如投资尽调、公关方案） | **袋袋 Profy** |
| 需要管理数字员工团队，强调治理和权限 | **QoderWake** |
| 需要本地部署、7×24 运行 | **QoderWake** |
| 需要视频生成（Seedance 2.0） | **天工超级智能体** |
| 需要企业 IM 集成（飞书/钉钉） | 天工（飞书）/ QoderWake（钉钉） |

## 七、总结

天工、袋袋、QoderWake 三款产品，覆盖了 AI 员工化的三种形态：

- **天工超级智能体**：云端 AI 团队平台——让 AI 干活的入口降到最低，打开浏览器就有一支团队
- **袋袋 Profy**：AI 专家市场——让专家的经验变成可交易的数字资产，一次封装终身复利
- **QoderWake**：本地数字员工平台——核心不是能力而是治理，让 AI 员工像真人员工一样可追踪、可管理

三者的共同趋势是：AI 正在从"工具"演变为"员工"。不再是你问一句它答一句的被动响应，而是有岗位、有记忆、能独立交付结果的生产力单元。而三条不同路线的出现，意味着这个市场正在分化为不同的需求层次——不是谁替代谁，而是不同场景选不同方案。
