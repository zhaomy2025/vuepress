**操作步骤**：
1. 右键 **此电脑** → **属性** → **高级系统设置**
2. 在 **性能** 框中点 **设置**
3. 切换到 **高级** 选项卡 → 在 **虚拟内存** 框中点 **更改**
4. 取消勾选 **自动管理所有驱动器的分页文件大小**
5. 选中 **C:** → 点 **无分页文件** → **设置**
6. 选中 **D:** → 点 **系统管理的大小** → **设置**
7. 点击 **确定** → **立即重启**
